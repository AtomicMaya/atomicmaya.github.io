As I may have mentioned in the past, I am currently an employed person.

This blog post will not go into the specifics of said employment, but rather go into a task that I was undertaking, specifically to integrate Snipe IT -- an inventory management system -- with our Azure tenant, mainly in the sense of syncronizing the users within Snipe with the Entra ID Directory Services, and provide seamless login to Snipe to view the assets one is allocated.

But this proved to be quite a headache, and as a result I wanted to write up the full process for public consumption, because it definitely had a few pitfalls which I had to overcome.


Requirements

Doing this obviously requires certain permissions on the Azure tenant, as you need the ability to create organization resources and set up entreprise applications.

Furthermore, it requires the permission and the authority to set up a service which costs, as Entra ID Directory Services is not free to run as a service.
